# Pulto Lua Obfuscator

A basic online Lua obfuscator for Roblox scripts. This project is currently in its **early stages**, with the core functionality working. Updates and improvements are being added regularly.

---

## Features

- **Byte Encoding** – Converts Lua code into escaped byte sequences for safer distribution.  
- **Randomized Junk Blocks** – Adds fake math operations and dead control flow for obfuscation practice.  
- **Wrapper Support** – Wraps your code in a functional `loadstring("...")()` call.  
- **Live Decrypt Tool** – Quickly decode byte-encoded Lua scripts.  
- **Basic UI** – Tabbed interface for obfuscation and decryption.  

---

## Usage

1. Enter your Lua code into the input box.  
2. Choose the obfuscation options you want:  
   - Byte Encode  
   - Fake Math Blocks  
   - Dead Control Flow  
3. Click **Obfuscate** to generate your encoded Lua script.  
4. Copy the output for use in your projects.  
5. Use the **Decrypt** tab to decode any byte-encoded scripts.
